# raspi
My play ground with Raspberry PI and sensors.


## Installation - server (on a raspberry pi / 0):

### Change into the directory

```shell
cd server
```

### Install dependencies

```shell
npm install
```

### Run the server

```shell
npm start
```

## Installation - client:

### Change into the directory

```shell
cd client
```

### Install dependencies

```shell
npm install
```

### Run the server

```shell
npm start
```
Now just point your browser of choice to localhost at port 3000.

```shell
http://localhost:3000
```
